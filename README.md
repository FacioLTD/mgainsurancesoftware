# MGA Insurance Software Website

A highly optimized website for MGA insurance software comparison, designed for both search engines and AI agents (LLMs that perform web searches).

## Features

### SEO Optimization
- **Comprehensive Meta Tags**: Title, description, Open Graph, Twitter Cards
- **Structured Data (JSON-LD)**: Article, FAQPage, Organization, WebSite schemas
- **Semantic HTML5**: Proper heading hierarchy, ARIA labels, semantic elements
- **robots.txt**: Optimized for search engines and AI crawlers
- **sitemap.xml**: Complete site structure for search engines
- **Canonical URLs**: Prevents duplicate content issues
- **Mobile Responsive**: Fully responsive design for all devices

### AI Agent Optimization
- **Structured Data**: Multiple JSON-LD schemas for better AI understanding
- **Semantic Markup**: Clear content structure for LLM parsing
- **FAQ Schema**: Structured FAQ data for AI agents
- **Clear Content Hierarchy**: Well-organized sections with descriptive headings
- **robots.txt**: Explicitly allows AI crawlers (GPTBot, Claude-Web, etc.)

### Performance
- **Optimized CSS**: Modern, efficient styling with CSS variables
- **Lazy Loading**: Images and content loaded on demand
- **Debounced Scroll Events**: Performance-optimized event handlers
- **Intersection Observer**: Efficient scroll animations
- **Minimal JavaScript**: Only essential interactive features

### Accessibility
- **ARIA Labels**: Proper accessibility attributes
- **Skip Links**: Keyboard navigation support
- **Focus States**: Clear focus indicators
- **Semantic HTML**: Proper use of HTML5 semantic elements
- **Screen Reader Support**: Optimized for assistive technologies
- **High Contrast Support**: CSS media queries for accessibility

## File Structure

```
mgainsurancesoftware/
├── index.html          # Main HTML file with all content
├── styles.css          # Comprehensive CSS styling
├── script.js           # JavaScript for interactivity
├── robots.txt          # Search engine crawler instructions
├── sitemap.xml         # XML sitemap for search engines
└── README.md           # This file
```

## Setup

1. Upload all files to your web server
2. Ensure `index.html` is set as the default page
3. Update canonical URLs in `index.html` if using a different domain
4. Update `sitemap.xml` with your actual domain
5. Submit sitemap to Google Search Console and Bing Webmaster Tools
